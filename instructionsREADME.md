# Free The Blob! (2025)

This repository contains the **Free The Blob!** project developed during the laboratory for the **2025/2026 edition** of the *Video Game Design and Development* course.

---

## Deadlines

**3rd November** Design document  
**7th December** Prototype Submission  
**11th January** Beta submission  
**24th February** Final Project submission  

## Requirements

### Source Code & Assets
The `Source` directory must include:
- Full game source code
- All used assets (music, SFX, graphics)
- Original/editable files when available  
  *(e.g., PSD, AI, XCF, FBX, etc.)*

---

### Executables
The `Executables` directory must contain:

| Platform | Required |
|---------|---------|
| Web (HTML) | Mandatory |
| Desktop (Windows or macOS) | At least one |

> All builds must support **Full HD (1920×1080)**.

---

### Images
- `Images/Screenshots/` → **min. 7 screenshots**, 16:9, 1920×1080
- `Images/Icon/` → icon used on itch.io (**347×275 px**)

---

### Gameplay Video
`GameplayVideo/` must include:
- **3–5 minute** gameplay video
- Format: **.mp4**
- Recorded with screen capture software

---

### Documents
`Documents/` must contain:
- Game description used on itch.io
- Final version of the **Game Design Document**

---

## Additional Material
This repository includes reference folders (screenshots, arcade cabin logos, asset sources).  
Students may add more supporting material if useful.

---

</small>_Trapped in a lab, born to escape. Free the Blob home!
